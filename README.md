# Markdown Syntax Guide

## 1. Headers
# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

## 2. Text Emphasis
This is **bold text** and this is __bold text__.
This is *italic text* and this is _italic text_.
This is ***bold and italic*** text.
This is ~~strikethrough text~~.

## 3. Lists
### Unordered Lists
- Item 1
- Item 2
  - Sub-item 2a
  - Sub-item 2b

### Ordered Lists
1. First item
2. Second item
3. Third item

## 4. Links and Images
### Links
Visit [Google](https://google.com).

### Images
![Logo Alt Text](https://example.com)

## 5. Blockquotes
> This is a single-line blockquote.
>
> This is a multi-line blockquote.

## 6. Code Blocks
Use `inline code` for short mentions.

```python
# Multi-line code block
def hello_world():
    print("Hello, World!")
```
